AvalancheOS X
AvalancheOS X is a new version of AvalancheOS that will constantly be updated and will be different from the regular AvalancheOS.

AvalancheOS X includes a new, simpler and modern UI that is very modular.

AvalancheOS X's apps are a lot more faster and will be a lot more customizable as Apps that are marked as "Container" are now native and do not rely on an iframe which made the OS a lot more slower.

AvalancheOS X is not a successor to AvalancheOS for the moment and is now available here!
